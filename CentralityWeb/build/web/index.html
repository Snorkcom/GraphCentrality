Анализ важности узлов

Выпускная квалификационная работа студента ОмГТУ специальности "Системный анализ и Управление" Баскова М.М.

Назначение программы

Программа позволяет:

1) загрузить граф с пользователями социальной сети в формате .net и отобразить его визуально;
2) рассчитать их центральность. Доступны 6 мер: центральность по степени связности, центральность по близости, центральность по посредничеству, центральность по собственному вектору, PageRank, HITS .
3) отобразить полученные значения центральности на графе, либо ранжированным списком.
